# fight cub: volume 0 [![Build Status](https://travis-ci.org/whoa-algebraic/game-off-2016.svg?branch=master)](https://travis-ci.org/whoa-algebraic/game-off-2016)

## Game synopsis

Genetically augmented super soldier bear must escape from the government lab that imbued her with her powers.

Hand-drawn Metroidvania style game made with Unity.
